# Security Policy

## Supported Versions

Only the latest release of NeuroMeshOSS is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within NeuroMeshOSS, please report it privately by emailing `security@neuromesh.local`. 

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to patch the issue before making a public announcement.
